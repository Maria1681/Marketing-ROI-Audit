# Marketing Campaign Performance & ROI Audit
**Strategic Data Analysis for $62M+ Global Portfolio**


##  Project Objective
This audit analyzes campaign efficiency across 12 months of marketing spend. The goal is to identify high-performing customer segments and provide data-driven recommendations for budget reallocation to maximize net ROI.

##  Key Analysis Workflow
1. **Data Integrity Audit:** Cleaned raw CSV data, handled missing values, and verified currency consistency.
2. **Segment Profiling:** Aggregated spend and conversion metrics to calculate weighted ROI per customer tier.
3. **Visualization:** Developed executive-level charts with customized styling (Navy/Professional branding).
4. **Insights:** Drafted strategic observations based on statistical trends in the data.
